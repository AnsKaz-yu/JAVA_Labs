Сборка приложения

Требования
- Установлена JDK 
- Установлен Git

Сборка
- Откройте командную строку или терминал
- Перейдите в каталог, в который вы хотите склонировать репозиторий
- Клонируйте репозиторий с помощью команды Git:
  git clone https://github.com/AnsKaz-yu/JAVA_Labs.git
- Перейдите в директорию Progect
- Скомпилируйте все файлы .java:
  javac *.java
- Проверьте, что появились файлы .class
