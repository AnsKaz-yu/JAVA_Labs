Запуск приложения

- Откройте командную строку (терминал) и перейдите в каталог проекта
- Запустите программу
  java Main
- Сделуйте инструкциям на экране

Кодирование файла

- Выберите режим кодирования (Code) и введите имя входного файла
- После завершения кодирования полученный файл будет находится в каталоге проекта и иметь название output.ark

Декодирование файла

- Выберите режим декодирования (Decode) и введите имя входного файла
- После завершения декодирования полученный файл будет находиться в каталоге проекта и иметь название output.ark

Информирование

- Выберите режим информирования (Info) и введите имя входного файла
- После завершения программы Вы увидите в консоли информацию о размерах исходного 
  и закодированного файла, а также степень сжатия и дерево кодирования
